# Ai Message Bot - AI消息守护机器人

## 产品简介

**AiMsgBot**是一款支持多平台的无人值守消息发送机器人程序，可集成以下聊天软件：

- 微信/企业微信

- QQ/企点

- 飞书

- 钉钉

- 以及更多...

通过本工具，您可以将自己的聊天账号转换为自动化机器人，通过API驱动执行消息发送、文件传输、图片分享等任务，实现业务流程自动化。

## 技术特点

- **合法合规**：非逆向工程实现，无侵入、无hook、无内存修改 

- **稳定性强**：支持长时间运行，无需人工干预

- **数据安全**：提供独立数据隔离，敏感场景可定制高级安全方案

- **多平台支持**：统一API接口，适配不同聊天软件

- **使用便捷**：一键部署，任务栏后台运行

## 使用步骤

### 1. 准备工作

- 准备一台**专用Windows电脑**或虚拟机（性能满足运行目标聊天软件）

- 确保电脑**持续联网**，稳定运行

- 准备一个**未被官方封禁/禁言**的聊天账号（非常重要！） 

### 2. 部署流程

**步骤1：下载安装**

- 访问下载地址：[AiMsgBot官方下载](http://aibot.fx-i.cn/aibot.zip)

- 下载并解压主程序及对应平台配置文件

**步骤2：环境准备**

- 在电脑上安装并登录目标聊天软件

- 打开任意聊天窗口（钉钉用户注意：群聊和单聊界面差异导致机器人只能支持一种消息方式）

**步骤3：启动服务**

- 运行AiMsgBot程序（可在任务栏找到图标）

- 程序自动后台运行，开始监听API指令

### 3. API使用指南

获取机器码：点击程序界面的"关于"按钮，复制显示的机器码（mcode）

**API基础信息**

- 请求地址：`http://msg.fx-i.cn:85/aibot/msg/?act=sendmsg`

- 请求方式：POST

- 请求头：`Content-Type: application/json`

## 注意事项与限制

1. **账号安全**：

    - 严禁使用已被官方处罚（封号/禁言）的账号 

    - 建议使用专门的工作账号，避免影响个人账号

2. **功能限制**：

    - 钉钉用户注意：机器人只能支持群聊或单聊一种消息方式（因界面差异）

    - 文件传输支持常见格式，大小限制根据平台不同（一般不超过20MB） 

    - 部分平台可能对消息频率有限制，避免高频发送导致账号异常

3. **合规声明**：

    - 本工具**仅可用于合法业务场景**，如自动通知、业务提醒、数据报告等

    - 严禁用于垃圾信息发送、欺诈、侵犯隐私等违规行为

    - 一经发现违规使用，将立即封停账号并追究相关责任
